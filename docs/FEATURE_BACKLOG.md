# 功能待办清单

基于 Todoist 和 Todo清单（evestudio.cn）竞品调研整理，按优先级排序。

---

## P0 — 核心缺失

### 重复任务
- 支持每天/每周/每月/自定义重复规则
- 可设置重复次数（永久或 N 次）
- 完成后自动生成下次实例
- 修改时可选范围（本次 / 所有未来）

### 番茄钟（专注计时器）
- 可配置工作时长和休息时长
- 白噪音库（多类型可选）
- 沉浸模式（防烧屏、随机倒计时）
- 专注记录历史（关联事件名称 + 累计番茄数）
- 完成提示音 + 推送通知
- 番茄统计趋势图

---

## P1 — 高价值优化

### 事件工作量标记
- 工作量三档：一般（灰）/ 中等（橙）/ 较高（红）
- 事件左侧色条标识
- 区别于优先级（重要程度 ≠ 工作量）

### 子任务增强
- 父任务显示子任务完成进度条
- 所有子任务完成时自动勾选父任务

### 数据复盘升级
- 昨日小结：昨日工作量、专注时长、7 天趋势
- 周报：本周事件达成数、番茄收成、最佳工作日
- 完成率趋势图
- 累计事件总数 / 累计番茄总数
- 时间掌控评分

---

## P2 — 体验提升

### 自然语言输入
- 输入框支持 "明天买牛奶 #购物 !高" 格式
- 自动解析日期、标签、优先级
- 可扩展语音输入（长按触发）

### 分享列表
- 列表 / 统计导出为图片
- 分享到其他应用
